# 目标后端 UI 设计要点

基于参考图分析，目标 UI 风格具有以下特征：

## 整体配色
- **背景色**：纯黑色 (#000000 或接近黑色)
- **侧边栏背景**：深灰/黑色，与主内容区一致
- **卡片背景**：深灰色带细微边框 (类似 #1a1a1a 或 #111)
- **文字颜色**：白色为主，灰色为辅助文字
- **边框**：深灰色细边框 (类似 #333)

## 侧边栏设计
- 标题：「Orpheus Admin」白色文字
- 导航项：图标 + 文字，灰色常态，选中状态有浅色背景
- 底部：「View Site」链接 + 用户头像区域
- 用户区域显示：头像（圆形灰色背景）+ 用户名 + 角色标签

## 主内容区
- 页面标题：大号白色文字
- 副标题/描述：灰色小号文字
- 按钮：白色背景 + 黑色文字，圆角设计
- 卡片：深色背景，圆角边框，内部文字为白色/灰色层级

## 按钮风格
- 主按钮：白色背景，黑色文字，带图标
- 操作按钮：图标按钮，hover 时显示

## 关键差异（与当前设计对比）
1. 当前使用浅色背景 (slate-50)，目标使用纯黑/深色背景
2. 当前卡片为白色，目标为深灰色
3. 当前文字为深色，目标为白色/灰色
4. 整体需要从浅色主题转换为深色主题
